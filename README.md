## 📖 Despre Proiect
Acest proiect este o aplicație interactivă creată în Python pentru a testa, analiza și vizualiza performanța diferiților algoritmi de sortare. Scopul nostru este de a oferi un benchmark clar între algoritmi simpli (ex. Bubble Sort) și cei eficienți (ex. Quick Sort, Merge Sort).

Aplicația măsoară:
* Timpul de execuție.
* Numărul de comparații și interschimbări (swaps).
* Comportamentul pe diferite seturi de date (liste generate aleator, liste deja sortate, liste sortate invers).
